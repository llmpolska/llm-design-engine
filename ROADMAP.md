# Roadmap

## Available now

- Local-first brief, directions, generation, refinement, approval, preview, brandkit, lint, and export commands.
- Deterministic mock reasoning and image providers.
- OpenAI-compatible reasoning and image adapters.
- JSON-safe design AST with Markdown artifacts.
- Deterministic HTML/CSS/SVG preview renderer.
- Vue studio with brief, direction, specification, preview, brandkit, assets, lint, and export views.
- Public foundry landing page with a GastroOps case study.

## Next

- Team review comments and versioned direction comparisons.
- Provider configuration UI with request logs and retry boundaries.
- Additional deterministic renderer primitives and screenshot baselines.
- Read-only MCP resources for design artifacts.
- Repository scanner that reports implementation coverage without rewriting code.

## Later

- Autonomous image-to-code experiments behind an explicit opt-in package.
- Full visual implementation verification with human-review checkpoints.
- Multi-page route graphs and component-level artifact exports.
- Shared design libraries authored from approved project-specific directions, never premade themes.

The deferred items are intentionally not presented as MVP features.
