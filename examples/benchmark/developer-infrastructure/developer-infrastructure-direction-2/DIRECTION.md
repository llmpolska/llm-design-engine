# Creative direction

- id: developer-infrastructure-direction-2
- name: Make / Health 2
- metaphor: health as a ledger of make
- visual narrative: Deploy enters as context, concentrates around make, then resolves through an explicit next action. This direction gives make the first move and health the proof.
- composition: document
- typography: Use typographic contrast to mark incidents and let make set the reading pace.
- materials: indexed health sheets, deploy marks, make surfaces, trace traces
- interaction: incidents changes emphasis in place while preserving the original route.

## Domain elements

- make
- health
- deploy

## Avoid

- unrelated decorative gradients
- generic dashboard mockups
- feature-card grids
- floating glass panels
- abstract blobs
