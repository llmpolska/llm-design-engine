# Composition plan

- opening: document
- density: balanced

## Reading path

- study context
- evidence decision
- organize evidence
- confirmation

## Focal hierarchy

- study
- evidence
- organize
- recovery

## Sections

## Study entry

- layout: vertical field
- focal point: study
- Start with Preserve uncertainty without losing forward motion
- Anchor the first read to study

## Evidence lane

- layout: sequential action track
- focal point: evidence
- Make evidence the dominant transition
- Keep adjacent context visible

## Organize context

- layout: layered record
- focal point: organize
- Show relationships around organize
- Use detail as evidence, not decoration

## Responsive transformations

- desktop keeps the reading path lateral
- tablet reduces overlap before reducing type
- mobile stacks by action order and preserves source labels
