# Narrative

Study enters as context, concentrates around organize, then resolves through an explicit next action. This direction gives study the first move and sources the proof.

# Composition

## Study entry

- id: educational-research-entry-0

- layout: horizontal reading path
- focal-point: study
- alignment: start
- Start with Preserve uncertainty without losing forward motion
- Anchor the first read to study

## Compare lane

- id: educational-research-action-0
- height: min(82svh, 760px)
- layout: anchored action surface
- focal-point: compare
- alignment: start
- Make compare the dominant transition
- Keep adjacent context visible

## Organize context

- id: educational-research-context-0
- height: min(82svh, 760px)
- layout: evidence shelf
- focal-point: organize
- alignment: start
- Show relationships around organize
- Use detail as evidence, not decoration
