# Creative direction

- id: boutique-hotel-direction-1
- name: House / Rooms 1
- metaphor: house as a route through rooms
- visual narrative: House enters as context, concentrates around coordinate, then resolves through an explicit next action. This direction gives house the first move and rooms the proof.
- composition: route
- typography: Use typographic contrast to mark resolve and let house set the reading pace.
- materials: marked house surfaces, house marks, coordinate surfaces, orchestrate traces
- interaction: resolve changes emphasis in place while preserving the original route.

## Domain elements

- house
- rooms
- coordinate
- guest

## Avoid

- unrelated decorative gradients
- generic dashboard mockups
- feature-card grids
- floating glass panels
- abstract blobs
